- 查看当前安装的java版本：  
  `java -version`

- install OpenJDK：  
  `sudo apt install default-jre`  
  verify:  
  `java -version`  

- install jdk:  
  `sudo apt install default-jdk`  
  verify:  
  `javac -version`
