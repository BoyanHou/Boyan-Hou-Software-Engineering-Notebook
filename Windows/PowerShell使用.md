# PowerShell使用  
[ref:web](https://programminghistorian.org/en/lessons/intro-to-powershell#quick-reference)  
- 不区分大小写  
- PowerShell 常见命令自带bash alias  
  - 例如:  
  
    | PowerShell | Bash |  
    | ---------- | ---- |
    | `Set-Location`, `sl` | `cd` |
    | `Get-ChildItem`, `gci` | `ls` |
    | `mkdir` | `mkdir` |
    | `Copy-Item` | `cp` |   
    | `Move-Item` | `mv` |  
    | `Remove-Item` | `rm` |
